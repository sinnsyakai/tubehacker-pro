# 🎬 TubeHacker Pro

YouTube動画を分析し、黄金パターンを抽出するAIツール

## 機能

- 🔍 **動画分析**: YouTube動画のタイトル、サムネイル、構成を分析
- 📊 **共通項抽出**: 複数動画からパターンを抽出
- 💡 **企画生成**: 分析結果に基づいた企画案を生成
- 📝 **台本生成**: 完全な台本を自動生成

## 使い方

1. APIキーを設定（⚙️設定ボタン）
2. 動画URLを入力して分析
3. 共通項を抽出
4. 企画・台本を生成

## 必要なもの

- Gemini API Key ([取得はこちら](https://aistudio.google.com/app/apikey))
